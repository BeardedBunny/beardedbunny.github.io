Qui es-tu ?

Je m'appel Alexandre, et j'ai pour but de devenir developpeur web !



Pourquoi un site sur mes passions ?

Pour réaliser mon objectif de devenir développeur web, il faut forcement apprendre les differents langage du web ... mais ce n'est pas tout. Il faut surtout coder, coder, et encore coder.

Je suis peut-être un peu vieux jeu du haut de mes 25 ans mais je pars du principe que c'est en forgeant qu'on devient forgeron. Du coup, je me suis dis, pourquoi pas faire un site sur moi pour commencer ? Au moins, je connais bien le sujet (je pense ? ) et c'est un bon entrainement pour commencer mon périple !

D'ailleurs, afin de commencer doucement, j'ai utliser les langages suivants pour ce site : Html - CSS - Boostrap - Jquery (pas beaucoup) et Git.

Du coup, voici le tout premier site de mon portfolio de developpeur !
